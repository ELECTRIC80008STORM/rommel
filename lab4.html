¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Java es un lenguaje de programación orientado a objetos, originalmente nació con la intención de tener poca dependencia de implementación. Por otro lado, JavaScript surgió para agregar programas y funcionalidades extra a páginas web. La mayor similitud entre las dos es el nombre, por un lado Java es un lenguaje compilado, y JavaScript es interpretado. (Cerón Soria M., 2023).

¿Qué métodos tiene el objeto Date? (Menciona al menos 5)

getFullYear(): Permite obtener el año como un dígito de cuatro números.
getMonth(): Permite obtener el número de mes como un número del 0 al 11.
getDate(): Permite obtener el día como un número del 1 al 31.
getDay(): Permite obtener el día de la semana como un número del 0 al 6.
getHours(): Permite obtener la hora como un número entre 0 a 23.

¿Qué métodos tienen los arreglos? (Menciona al menos 5)

push(): Permite añadir elementos al arreglo.
unshift(): Permite añadir un elemento al inicio del arreglo.
pop(): Permite eliminar el último elemento del arreglo.
shift(): Permite eliminar el elemento del inicio del arreglo.
concat(): Permite añadir múltiples arreglos.
find(): Permite saber si un elemento existe dentro de nuestro arreglo.

¿Cómo se declara una variable con alcance local dentro de una función?

Se puede declarar ya sea con let o con const, la diferencia es que una variable const no puede ser reasignada, mientras que una let sí.

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Qué pueden ser accedidas desde fuera del cuerpo de la función, si bien en ciertos casos podrías beneficiarte de ello, suele ser riesgoso pues da la posibilidad de romper el sistema si se maneja de una manera no planeada por fuera de la función, lo que añade otro lugar como posible causa de errores.
